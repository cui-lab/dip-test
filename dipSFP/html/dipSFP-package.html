dipSFP-package {dipSFP}R Documentation

SFP detection based on dip statistic

Description

Given a matrix of microarray data, SFPs are detected based on the dip statistic. The null distribution of dip statistic is estimated by smooth bootstrap algorithm. P values are adjusted using false discovery rate criteria for FDR control.
